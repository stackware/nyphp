Apache Community Incubator Project

PHP and MySQL Open Source Projects

April 10th, 2005

A new Apache project, providing an entry path to the Apache Software Foundation for projects and code bases wishing to become part of the Foundation's efforts. Code donations from external organizations and existing external projects wishing to move to Apache will enter through the Incubator. A large part of the Incubator's effort will be devoted to providing documentation on how the Foundation works, and how to get things done within its framework. As a consequence, it is expected that the Incubator will also become a reference for new contributors to any of the Apache projects.

Visit: http://incubator.apache.org/ for more information.
