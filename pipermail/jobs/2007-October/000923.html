*Job Description:*

The ideal candidate will work closely with the project manager on the
development of a CMS to handle multimedia production. This includes
developing an internal tool kit that produces a flexible media player that
allows a user to mange the media assets (player templates, video, audio,
skins) and a publishing tool for textual content for a web site. The code
developed needs to be efficient and created with documentation. The
candidate needs to be resourceful and willing to take initiative. We are
looking for a person who can identify and learn new web technologies. This
is a NYC based opportunity.



*Skill Set*:

·          Proficient in LAMP development (Linux, Apache, MySQL, PHP)

·          Drupal development experience a strong asset

·          Intermediate JavaScript

·          Knowledge of XML or creating dynamic play lists to serve media

·          Familiarity with ActionScript 2 an asset, although ActionScript 3
preferred.

·          Knowledge of streaming media formats and encoding rates

·          Excellent in solving complex technical and business problems.

·          Excellent communication skill.

·          Ability to work within a team setting

·          Health care business knowledge would be an asset



*Key Responsibilities:*

·          In certain cases finalizing the business requirements and finding
the technical solution

·          Develop, Support & maintain applications

·     Understand the new requirements; develop and unit test the code to
meet the specs/ requirements.
