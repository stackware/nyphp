Please post the following job:

The International Trademark Association (INTA), a trade association in midtown east NYC, is seeking an in-house web developer to manage existing applications and build new ones.  INTA is in the process of re-launching its website on Mambo/Joomla.

Existing applications which are currently outside of Mambo/Joomla include:
*	User authentication and account management.
*	Existing and future online publication applications.  (These are complex outline structures written in PHP and mySQL that allow INTA members to create and edit content, in-house editors to copy-edit, and INTA staff to present to the web.)
*	E-commerce, an image library and education program course materials and brochures.

The ideal candidate will have:
*	Computer science degree
*	4-5 years experience in web development
*	2 years web development experience with Mambo
*	Solid ability with PHP and mySQL along with facility writing SQL queries
*	Ability to work well with non-technical staff on content-driven projects

Because the web developer will assist in determining the feasibility of integrating existing applications into the new framework, experience with Mambo/Joomla is preferred.  Please send a cover letter indicating salary requirements and history with resume to: webdev at inta.org.
