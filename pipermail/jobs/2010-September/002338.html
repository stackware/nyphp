University division is seeking a junior/entry-level programmer to help create and maintain Web applications with PHP, MySQL and Javascript (jQuery), as well as research emerging Web technologies for our community of users (graduate students, faculty and staff).

This is a part-time job and but may become a full-time job in the future. Starting rate is $18/hr.

*Responsibilities*
Help maintain and develop Web applications; troubleshoot Web pages through the OmniUpdate Content Management system; manage and maintain website integration with MySQL database and  Salesforce.com; prepare digital photos for display online; provide support for website redesign and other administrative tasks as needed. 
  
*Qualifications*

Bachelor’s degree in computer science or similar discipline.  Currently enrolled computer science students with strong work experience will be considered.

_Required Skills_: Proficiency with  HTML, CSS, PHP, Javascript, MySQL, RSS/XML

_Preferred_: jQuery, Perl, Ajax, Photoshop and Web design software (e.g. Flash), XSL

_Big Plus_: Knowledge of and emerging Web technologies (for example Google Maps APIs, Facebook Apps); mobile web development; Linux; experience with Salesforce Apex code (similar to Java), PHP Web Services toolkit and Visualforce (http://wiki.apexdevnet.com/index.php/Apex_and_Visualforce)

Please send cover letter and resume to Paul Tepper at ptepper at nyu.edu.
