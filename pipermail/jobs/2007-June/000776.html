Job Description:
Are you passionate about online video and the communities that grow
around it? Are you always looking at websites and mentally looking at
the code driving them? If you are an expert at developing
high-performance, MVC standards-compliant interactive web applications
using Linux/Apache/MySQL/PHP, we want to talk to you!

We are looking for a developer specialized in interpreting interface
design and coupling it with a backend framework to present dynamic and
efficient web applications. The ideal candidate has the ability to write
and understand code as well as being able to work under pressure in a
dynamic environment. The candidate should be comfortable facing
development challenges with enthusiasm while also researching and
proposing new technologies in their field.

Minimum Job Requirements:

*	3-5 years experience developing code for standards-compliant,
MVC web applications. 
*	PHP (OO programming a plus). 
*	MySQL query construction. 
*	XHTML, CSS and Javascript. 
*	Linux OS shell environment and utilities. 
*	XML and RSS standards.

Preferable Job Qualifications:


*	Experience building Web communities. 
*	Experience and understanding AJAX methodologies. 
*	Flash and Actionscript. 
*	Digital video technologies, including encoding, distribution and
streaming.
