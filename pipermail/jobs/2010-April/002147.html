      Job Responsibilities
     
      We are looking for a Web Developer to support the U.S. Merchant Marine Academy at Kings Point, NY.

      The purpose of this position is to create and renew content and navigation patterns quarterly for the following web sites: the Academy's general web site, campus intranet web site, library web site, general/campus athletics web site, general/campus waterfront activities web site, and other portal web sites.

      The candidate will create and maintain web site graphics, narrative content, site and page structure reflecting Web 2.0 standards for XHTML, Javascript, and cascading style sheet (CSS). All web sites shall comply with federal accessibility standards, transmit and receive personal identification information using SSL, and support the most recent versions of web browsers from Microsoft, Firefox, Apple, and Opera. Where possible, web pages shall display and properly operate on personal computers and cellular phones equipped with web browsers.
