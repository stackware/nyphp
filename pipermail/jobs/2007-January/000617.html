LAMP Developer

New York University’s Office of Web Communications seeks a highly  
skilled LAMP Web Developer for a short-term (roughly one month)  
contract position beginning immediately. The applicant will work in  
coordination with existing development and production staff to  
implement database-driven Web applications, maintain existing  
applications, and field minor development-related queries.

Applicant should have 4+ years' experience developing Web  
applications, managing website content and file system permissions,  
knowledge of security procedures, and creating and modifying scripts;  
Excellent planning, analytical, problem solving and communication  
skills; Knowledge of web programming languages, including PHP, Perl,  
SQL, CSS, XML, Javascript, as well as Apache and UNIX development  
environments.

In addition to managing the home page, the Office of Web  
Communications operates as a boutique studio-style resource to the  
entire university. The Web Developer works with two producers, a  
designer, and a project manager under the supervision of the Director.

If you are interested, please forward me your resume and salary  
requirements.

Joel Leimer
Director, NYU Web Communications
