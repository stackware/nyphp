JOB SUMMARY
Responsible developing responsive websites and web applications on open-source platforms for B2B media firm. The Web Developer will have a strong knowledge in software architecture and design and have experience working with open-source projects.

ESSENTIAL JOB FUNCTIONS
The responsibilities listed below are representative of the nature and level of work assigned and are not necessarily all-inclusive.  Each incumbent in this job:

  o Maintain and enhance editorial content management and display
  o Research, document and implement development requirements and specification.
  o Create websites and web applications on open-source platforms
  o Ability to work independently while building on the contributions of others
  o Create optimal workflows for internal administrative users
  o Design performance and maintainable databases.  Optimize queries to use indexes and key structure.
  o Source Code Management: use SCM to track changes, create branches, and merge patches between stable and release branches.

JOB REQUIREMENTS
Minimum requirements that an individual should possess to carry out the responsibilities of the job.

  o Bachelor's Degree in Computer Science, or related field
  o Experience with building and supporting CMS
  o Possesses significant knowledge of client server and internet systems architectures.
  o Creates well designed, reusable code on a reliable schedule.
  o Understands specific browser compatibility issues.
  o Understands the benefits of the various design methodologies, functional, and object oriented environments.
  o Excellent oral and written communication skills

COMPENTENCIES
Work behaviors that drive successful performance on the job.

  o Assists in the facilitation of team and client meetings.
  o Delivers informative, well-organized presentations.
  o Understands how to communicate difficult/sensitive information tactfully.
  o Generates enthusiasm among team members.
  o Manages the process of innovative change.

REQUIRED TECHNOLOGIES
  o PHP
  o JavaScript
  o CSS
  o XHTML
  o MySQL

EXPERIENCE WITH THESE TECHNOLOGIES A PLUS
  o .NET
  o Firebird
  o jQuery
  o memcached
  o Postgres
  o Prototype
  o Python
  o Ruby
  o Sphinx Search
  o Yahoo UI
