Full-time Web Developer for Design Studio (Drupal, PHP, HTML/CSS/JS) (SoHo)

Our boutique interactive design & development firm is looking for a 
strong, mid-level web developer. This position is for a full-time, 
on-site position ONLY.

The ideal person is looking to work in a creative and collaborative 
environment and is fluent with PHP, MySQL, Drupal development and 
theming, and should have a solid grasp of core front-end technologies 
(HTML, CSS, JavaScript, jQuery, AJAX, JSON). We’re looking for an 
outgoing person who thinks about their work in the larger business 
context, is a problem solver and creative thinker, proactively 
collaborative, and understands how to effectively manage their time to 
meet deadlines.

The core responsibilities for this position are the development of 
custom Drupal CMS and web applications as well as the development & 
maintenance of existing Drupal installations.

RESPONSIBILITIES
• Develop, write, test and debug code to match functional and/or 
technical specifications
• Drupal CMS set-up
• Modify and customize existing Drupal CMS modules
• Integrate front-end templates and Drupal theming
• Contribute UI detailing via JavaScript, jQuery and AJAX/JSON
• Produce documentation for the use of other web developers

REQUIREMENTS
• 3+ years of professional web development experience
• Strong knowledge of Drupal, PHP 5+ and MySQL
• Writes concise, commented, maintainable, portable code
• Strong working knowledge of HTML, CSS, Javascript, jQuery, AJAX/JSON
• Experience developing web applications for multiple platforms and 
browsers.

APPLY
• Resume, cover letter & sample projects.

***NO TELECOMMUTING***
***NO RECRUITERS***
***WE'RE NOT LOOKING FOR DRUPAL "PARTNERS" BUT A FULL-TIME, ON-SITE 
EMPLOYEE***
