Publishing company seeks proactive front-end web developer with
HTML/PHP/Drupal skills to join fast-paced interactive team. Selected
candidate will have 3-4 yrs HTML/CSS/JS (very comfortable), 2 yrs Drupal
theming experience, and up to 2 yrs experience/exposure to PHP5. Looking
for a self starter who is deadline driven and possesses strong
communication & organizational skills.

 

Specific Duties and Responsibilities:

 

* Work with back-end developers, product managers and designers to
implement pixel perfect front-end and user interface requirements.

* Interact with clients and/or other departments during the development
cycle

* Stay abreast of the new front-end web development technologies;
integrate them with the company*s web sites to further enhance the
usability and efficiency of the sites as necessary.

* Work within tight deadlines while exhibiting exemplary time management
and task prioritization

 

Position Qualifications:

 

* 3-4 years web development experience in a production environment

* 3-4 yrs HTML, CSS, Javascript. Applied knowledge of HTML5/CSS3

* 2 yrs Drupal 6 experience, including theme development (more info
below)

* 1-2 yrs PHP (light, front-end)

* Excellent knowledge of Photoshop, slicing assets from PSDs and
producing HTML markup

* Familiarity with Browser quirks, cross-browser testing, alternate
stylesheet

* Familiarity with general web development techniques/concepts: semantic
markup, seo techniques, xml sitemaps, htaccess files, robots.txt, etc

* Comfortable using source control tools (SVN)

* Familiarity with open-source PHP frameworks, i.e. CodeIgniter/CakePHP

* Familiarity with open sourse javascript libraries (JQuery, GMaps,
SWFObject, etc) a plus

* Familiarity with MySQL (or other relational database) and/or basic SQL
knowledge a plus

* Familiarity with OO PHP, and PHP5, a plus

* Passionate about web standards

 

Drupal-specific skillset:

 

* General Drupal theme setup/config: regions, working with .tpl files
and their naming conventions, Blocks, Menus, Building Taxonomies,
Building Content-types in CCK, Installing/configuring contrib modules
(pathauto, panels, views, admin menu, nodequeue).

* Experience creating custom Drupal themes from scratch (or using Zen
base theme): includes slicing assets from PSDs, writing CSS, HTML, and
light PHP programming in Drupal .tpl files.

* Build views for pages & blocks (using views module admin is fine) and
theme them

* Customize nodes and/or pages using pre-process functions or straight
tpl files (including menu, breadcrumb, user profile, tweaks for various
content-types, etc)

* Building panel pages via admin UI, theming them (panels module)

* Building views via admin UI, theming them (views module)

* Light jQuery (i.e. menu overlays, rollover interactions, etc), light
PHP

 

To apply, send resume and links to at least 3 production websites you've
worked on, along with detailed description of your role in the
development/production process.

 

Cheers,
