Web Developer
New York, NY (Midtown Manhattan)
6 month contract

Our client's Web Services Department is currently looking for a detail-oriented, hard-working, creative, responsible individual with experience building web applications for high-traffic media rich websites.  This is a full-time, on-site contract position in Midtown NYC.

*PLEASE NOTE*
Please note the hiring manager have expressed that referred candidates should have an executed aptitude test. Please submit the completed test with the candidate's resumes.
Responsibilities include
*    Develop new applications for Digital Development and other departments for public web sites (PHP, MySQL). Maintain and support existing web-based applications.
*    Help maintain Web Farm and hosting network in a heavily loaded production environment.  Manage external/internal web servers running on LAMP (Linux, Apache, MySQL, PHP) platform. Work with hosting provider and IT group to troubleshoot and resolve all issues.
*    Integrate web properties with systems; and various vendor systems: web statistics service, banner ad systems, video hosting and management service, Akamai EdgeSuite, and etc.
*    CMS workflows and template development, modification and testing; system configuration for Interwoven TeamSite CMS (XML, Perl)
*    Participate in project meetings with clients and work with other developers as well as project managers/business analysts to gather requirements.
Requirements
*    3+ years experience in the development of web based applications (PHP, MySQL, as well as other web technologies)
*    5+ years of experience in front-end technology with JavaScript, DHTML, CSS, and HTML.
*    Ability to multitask and handle multiple simultaneous projects with challenges deadline.
*    Ability to understand client's needs and cultivate partnering relationships. Outstanding analytic and problem solving skill, problem resolution and customer service.
*    Ability to establish goals and set clear expectations
*    Ability to prioritize and work tenaciously to achieve goals.
