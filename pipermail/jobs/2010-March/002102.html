truTV Web Services Department is currently looking for a detail-oriented, hard-working, creative, responsible individual with experience building web applications for high-traffic media rich websites.  This is a full-time, on-site contract position in Midtown NYC.

Responsibilities include:

- Develop new applications for truTV Brand Strategy and Digital Content group and other departments for public web sites (PHP, MySQL). Maintain and support existing web-based applications.

- Implement and maintain SharePoint-based corporate intranet portal, division sites and collaboration workspaces. Develop custom webparts using ASP.NET, update page layouts to user requirements. Develop and maintain custom internal applications (ASP, ASP.NET, MS SQL)

- Help maintain Web Farm and hosting network in a heavily loaded production environment. Manage external/internal web servers running on LAMP (Linux, Apache, MySQL, PHP) platform as well as Microsoft (MS IIS, ASP, ASP.NET, MS SQL, MS SharePoint). Work with hosting provider and IT group to troubleshoot and resolve all issues.

- Integrate web properties with Turner systems; and various vendor systems: web statistics service, banner ad systems, video hosting and management service, Akamai EdgeSuite, and etc.

- CMS workflows and template development, modification and testing; system configuration for Interwoven TeamSite CMS (XML, Perl)

- Participate in project meetings with clients and work with other developers as well as project managers/business analysts to gather requirements.

Requirements:

- Minimum 3 years of web development using LAMP stack (Linux, Apache, MySQL, PHP)
- Minimum 2 years of web development using ASP.NET, C#, ASP as well as experience with T-SQL, Stored Procedures, MS SQL Server. Development experience with Microsoft SharePoint technology.
- Minimum 2 years experience with Apache and Microsoft IIS web servers in production environment on Unix (Linux) and Windows platforms.
- Basic understanding of HTTP, TCP/IP

Preferred Experience:

- Minimum 1 year experience with XML, XSLT, HTML, JavaScript, CSS
- Knowledge of Interwoven TeamSite CMS templating, workflow development (XML, Perl)
- Experience with version control systems (CVS/Subversion)
- Basic understanding of Web Services, Streaming Media
