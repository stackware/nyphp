Full time hire in Elkridge, MD-( south of Baltimore, MD)

DATABASE ENGINEER

Job Function

Lotame is looking for a data mastermind to build the next generation of
internet marketing technology specifically geared towards social
networking websites like MySpace and Facebook. If you like to be
intimately involved in the design and development of a companys core
technology, want to own projects and make a real difference in a small
company, then Lotame wants you. 

 

Candidates for this position must be thoroughly knowledgeable in
database technologies, with specific knowledge of MySQL in a Linux
environment. 

 

The Day to Day

-Manage MySQL on Linux in Production/QA/Development environments

-Ensure database systems perform reliably

-Participate in the design and implementation of all aspects of the data
architecture, including data models, data flows, aggregations,
clustering, replication, and backup and recovery strategies

-Support team in troubleshooting efforts

-Participate in on-call rotation with other team members

 

 

Qualifications

-BS in a technical field or significant relevant experience

-3+ years of database experience 

-Proficient in SQL

-Experience in database performance tuning, backup and recovery, system
architecture and design with MySQL

-Ability to write scripts to support database administration activities

-Experience working with very large data sets

-Experience with data sharding/partitioning is a major plus

-Experience with MySQL stored procedures, functions, and triggers

-Knowledge of multiple MySQL storage engines

-Self-motivated with strong problem solving and analytic skills

-Ability to handle multiple tasks

 

Notes

-          Lotame wants someone who has worked in a large organization
and understands how a large scale open source environment works. 

-          You necessarily do not need to know all the skills, but you
do need to understand how to logically describe the function

-          MySQL background a MUST HAVE!
