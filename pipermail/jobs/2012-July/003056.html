Drupal Developer

Flightpath is an award winning interactive agency headquartered in New York City's Flatiron district. We provide digital imperatives - web and mobile app development, social media services and search engine marketing - along with full-service creative and production capabilities. Our client list includes BMW, Cablevision, Goya Foods and Scholastic to name a few.

We are currently seeking an experienced Drupal developer with excellent JavaScript and CSS skills to work on a variety of web-based applications.

Responsibilities:
* Develop and maintain web sites using Drupal according to Drupal best practices
* Create Drupal themes based on design comps
* Help project managers understand development requirements.
* Configure Drupal for optimal performance
* Develop JavaScript/DHTML web components

Qualifications:
* A passion for Drupal and learning new technologies
* Experience building sites with Drupal and Drupal technologies such as Drush, Features, Views, and Rules
* A deep understanding of Drupal theming and modules
* Experience with PHP, MySQL, HTML and CSS, AJAX, and JavaScript
* Familiar with SVN and Git source control systems
* WordPress, ASP.NET, SQL Server and Facebook API skills are a plus
* Ability to work independently and as part of a larger team
* Must be professional, dedicated and conscientious
* Strong communication and organizational skills and attention to detail
* Ability to work independently and as part of a larger team and handle multiple assignments

If you'd like to be an integral team member in a growing digital marketing agency, please send resume, cover letter and salary requirements via email. No recruiters please.
We offer Competitive salary, benefits including health and 401k, and a creative, entrepreneurial atmosphere.

Please send resume, cover letter, URLs and salary requirements.
