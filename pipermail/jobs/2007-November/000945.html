FREELANCE LAMP MASTER - ON-SITE

Ultrastar.com is looking for on-site PHP / MYSQL 5 master for a 6 - 10 week project. This is full-time in-house contractor position. Your job will be to help build and integrate a new social network platform as well as another large-scale social-wiki-based application.

Overview:  
Candidate will work with the in-house product development team to create new social-based applications, optimize existing code and queries. Some work with memcached may also find its way onto your plate. 

SKILLS
 Required: 
  100% OO PHP 5 (inheritance based class systems)
  MYSQL 5 (stored procedures and triggers)
  Web 2.0 / Social Networking application development experience
  Must work well in a team setting and fast paced environment 
 
 Bonus: 
  Knowledge of AJAX, CSS and JavaScript
  Interface design sensibilities
  Experience with memcached
 

Compensation: 
Hourly

How to Apply: 
Send cover letter, hourly rate, and online samples of work to jobs { at } ultrastar.com. Recipients without this information will not be considered. This is an IN-HOUSE position. Do not apply to work off site.
