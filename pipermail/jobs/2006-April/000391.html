Web Developer


Description:


The Web Developer will be involved in the development and maintenance of
online publications. The job requires on-site collaboration with a web
designer as well as the ability to research issues and solve problems on
one's own. The developer must be willing and able to work in a heterogeneous
web environment. This position will report to the VP of Technology.


Qualifications:


*         3-6 years web development experience

*         Proficiency with HTML/CSS and JavaScript

*         An ability to work with designers to implement efficient web
templating systems.

*         Knowledge of the technical challenges in building an
HTML/CSS/JavaScript site experience that supports all currently active
browsers.

*         Content Management Systems (CMS)

*         Programming/Scripting Languages: PHP, Perl, Ruby

*         Experience in the server side scripting: PHP

*         Basic UNIX skills (Linux, Solaris) desirable; Simple Unix shell
commands, telnet, ftp. and Shell scripting, bash, tcsh, or ksh.

*         Operation and design of Relational Databases (RDBMS): MySQL, MS
SQL, and/or Firebird

*         Configuration and maintenance of Web Servers  (IIS, Apache Tomcat)

*         Familiarity with version control systems: Subversion and CVS

*         Must be self motivated and effective working in a team environment

*         Strong communication skills, will gather requirements and interact
with non-technical editorial staff

*         Documentation of code and process, technical writing

*         A desire to learn new technologies and share knowledge


Duties:


*         Develop and maintain websites and services

 

Please submit your resume to jhaddock at wbi.com.
