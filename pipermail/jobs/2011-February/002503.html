*Web Developer (PHP/Drupal)*

Fulltime and/or Freelance Web Developer - (Drupal, PHP, Javascript, MySQL)

Our boutique-sized interactive design & development firm is looking for 
a strong, senior-level web developer to work with both our design and 
development teams.

The ideal person understands the importance of working in a 
deadline-driven environment, is fluent with PHP, MySQL, Drupal 
development and theming, and should have a solid grasp of core front-end 
technologies (HTML, CSS, JavaScript, jQuery, AJAX). We're looking for a 
mature, reliable, good natured person who can think broadly about their 
work in the larger business context, is highly creative, and understands 
what "done" means. The ideal candidate is able to work well with 
creatives and able to manage one's time in order to meet deadlines.

RESPONSIBILITIES
. Develop, write, test and debug code to implement the backend 
functionality, UI design, and all functional specifications for websites 
and web applications
. Drupal CMS set-up
. Modify and customize existing Drupal CMS modules
. Integrate front-end templates and Drupal theming
. Add UI detailing and improvements via JavaScript, jQuery and AJAX
. Produce documentation for the use of other web developers
. Writes concise, commented, maintainable, portable code

REQUIREMENTS
. 4+ years of professional web development experience
. Expert knowledge of Drupal, PHP 5+ and MySQL
. Strong working knowledge of HTML, CSS, Javascript, jQuery, AJAX
. Experience developing web applications for multiple platforms and 
browsers.

PLUS:
. Configuration of MySQL database and Apache web server
. UNIX/Linux administration

APPLY
. Resume with a well-written cover letter

COMPENSATION
Commensurate with experience
