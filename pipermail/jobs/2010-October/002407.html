Senior Front End Web Developer - Scholastic.com

 

The eScholastic division of Scholastic, Inc is seeking a full time
Senior Front End Web Developer to oversee and manage all front end
development work for the Scholastic.com web site. This is a unique
opportunity to play a key role in the revamping of a large web site for
a major publishing company. The person filling this position will be
responsible for the css/html architecture of the entire Scholastic.com
web site.

 

JOB DESCRIPTION

 

The Senior Front End Web Developer, who will work closely with senior
eScholastic Technology staff, will assume the following duties:

 

Establish and enforce HTML/CSS/Javascript coding standards for all
Scholastic.com sites

Create a CSS driven template system for new and existing site content.

Develop an incremental strategy to clean up/standardize existing sites
to meet standards

Lead a team of front end developers in all project and maintenance work
on the web site

Collaborate with the creative and technology teams to bring fresh ideas
and innovation to our web presence

 

REQUIRED SKILLS

 

Bachelor's degree in CS, Information Technology or related field; or an
equivalent combination of skills, training and experience. 

Expert HTML/CSS/Javascript coding abilties - including the latest
standards (HTML5, CSS3) and Web 2.0 / AJAX

Experience in creating or working with html/css/javascript coding
standards for large web sites

Experience with PHP based CMS systems  (Worpress, Drupal Concrete5),
developing themes, etc

Experience using version control systems (svn, git, mercurial)

Experience using web page scripting languages (php, ruby, python)

Knowledge of Testing/QA processes

Strong written and oral communication skills - ability to write
technical documentation.

Strong management skills; ability to lead and motivate teams of
developers in executing large projects

 

Please forward resume and salary requirements to agresens at scholastic.com
