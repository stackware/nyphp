The Lead Web Developer will be responsible for the team creating the consumer experience for Hasbro’s consumer facing online entertainment destination for TheHub Television network, hubworld.com. The lead web developer will report to the Director Engineering and Development, and work closely with other developers, DBAs, SysAdmins, and online product 



Responsibilities: 


    • Plan, build and maintain custom content management solutions using Java and PHP. 
    • Assist in the configuration and development of custom, commercial and open source content management systems. 
    • Coordinate with consultants and vendors on the development of custom tools as well as off-the-shelf software. 
    • Work with business clients to ensure that applications fit the needs of US and international users. 
    • Manage multiple web projects simultaneously on a daily basis. 
    • Day-to-day maintenance of respective brand web sites and supporting applications. 


R equirements: 

    • 7+ years of web development, 2+ years in a team leader position

    • Experience working with MVC CF frameworks 
    • Experience writing advanced SQL queries and working with MS SQL or similar servers. 
    • Familiarity with Alfresco a plus 
    • Endeca experience a plus 
    • Experience with various modern development methodologies, waterfall through agile. Scrum experience a plus 
    • Localization experience preferred 
    • Familiarity with the Internet and current trends in web technology, and the desire to learn appropriate new technologies. 
    • Familiarity with both Unix based server environments, knowledge of Windows a plus 
    • Ability to handle multiple projects concurrently 
    • Solid oral and written communication skills. 
    • Technical writing and editing skills desirable (e.g., technical specifications, help documentation, product documentation) 
    • Source code samples and URL samples are required. 
    • B.A., B.S. degree, or equivalent work experience preferred.

Sr. Software Engineer, Web 



The Sr. Software Engineer, Web will be responsible for the creating scalable web applications and web services supporting Hasbro’s consumer facing online entertainment destination. Hubworld.com, Hasbro’s new online entertainment destination for kids, featuring online games and videos from many of Hasbro and other brands featured on the new kids television network, TheHub. The Sr. Software Engineer, Web will report to the Director, Engineering and Development, and work closely with other developers, designers, DBAs, system administrators, and other product and marketing personnel. 

. 

Responsibilities: 

    • Hands-on development of systems, application and scalable web services supporting the hubworld.com entertainment web site (online games, videos, other entertainment content) 
    • Architect and develop scalable web applications, using modern development methodologies 
    • Development tasks will include a mix of designing, coding, integrating, testing, tuning, debugging, documenting, peer-reviewing and support. 
    • Assist in the configuration and development of custom, commercial and open source content management systems and other production web systems 
    • Manage multiple projects simultaneously 




Requirements: 

    • Proven track record of success in developing web applications and infrastructure for scalable consumer web sites, experience with online entertainment sites (online games, videos, etc) a plus 
    • Ability to define and implement new web based technology solutions and enhancements across multiple web environments to deliver maximum value for our business partners. Excellent technical leadership skills and ability to manage a diverse set of stakeholders. Technical skills should include 
    • 8+ years of experience developing web applications, with a solid understanding of Internet technologies and architecture 
    • 5+ years Java, PHP, or similar web application development language experience 
    • 5+ years experience with SQL database query development work, designing database schemas, tuning applications and databases 
    • Experience programming and integrating web systems components such as content management systems, ad serving systems, content distribution networks, etc. 
    • Technical expertise in Web architecture and design principles, Web Services / SOA, etc. 
    • Solid understanding of middleware and integration technologies 
    • Knowledge of XML, XSL, XSLT, Javascript, SOAP/REST, HTML and HTTP
