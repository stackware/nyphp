We would like to post the following job on the site:

 

The Web Developer is responsible for maintaining front-end code for
high-volume, high-traffic, end-user facing websites. Collaborating with
Engineering, Product, Design, Marketing, QA, Operations and Editorial teams,
the Web Developer implements and maintains final, production-quality code
directly within site HTML/CSS/jQuery/PHP code base.  The Web Developer will
be expected to work with version control systems. 

 

Required Technical Qualifications:

 

* Expert knowledge of Web Standards; the DOM; cross-browser & cross-platform
(X)HTML, JavaScript and AJAX

* Expert, in-depth knowledge of CSS positioning, hierarchy and (X)HTML/CSS
architecture

* Strong PHP skills

* Working knowledge of jQuery

* Working knowledge of Linux

*   Working knowledge of build systems

 

Required Competencies:

 

* Ability to work as part of a close-knit, cross-functional team with both
technical and non-technical team members

* Excellent communication, problem-solving, and organizational skills

* Excellent attention to detail and a high-level of accountability

* Ability to estimate development schedule for owned features

 

Extra Credit:

 

* Experience with LAMP, Flash/Flex

*Working knowledge of .NET and MVC frameworks

*Experience designing and building mobile applications for iPhone, Android,
Web OS, or Blackberry platforms

* Knowledge of Agile and/or similar development methodologies

 

 

 

Thanks!
