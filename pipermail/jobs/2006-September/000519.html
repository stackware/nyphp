** Please only developers who live in the New York Metro area and can

Software engineer working on LAMP based client/server software

applications as well as troubleshooting technical problems.  

Day to Day responsibilities include:
* Develop, test, and maintain high-quality, secure database web
applications  
* Supervise and mentor more junior programmers  Interact with web

QUALIFICATIONS: 
* Bachelor of Science Degree plus five years of experience in software
development, or equivalent combination of education and experience.
* Strong demonstrated work experience and expertise in disciplined
software development approaches, including test-driven development,
refactoring, and object-oriented analysis/design.

At least 3 years work experience and demonstrated expertise with all of
the following languages and tools:
* Professional standards-based web development using PHP, HTML, XML,
CSS, AJAX and JavaScript
* Advanced Smarty Template engine experience
* Java, C++, C# or another object-oriented programming language
* Advanced relational database system knowledge
* Some experience with Unix/Linux and Window Server administration
* Knowledge of security standards and techniques for web applications
* Expertise in the design, implementation, and deployment of
user-centric software, with sufficient attention and detail to usability
