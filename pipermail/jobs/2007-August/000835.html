UltraStar Entertainment: Web Application Engineer ( LAMP ) 

Ultrastar Entertainment seeks highly motivated Web Application Engineer. This is a key production roll for a growing company who will report directly to the Director of Product Development and Chief Information Systems Officer.

The candidate's primary objective is to maintain enhance and manage applications in a fast paced environment. The ideal candidate will be very familiar with PHP/Mysql Social Networking Applications and billing (payment) systems.

This is an exciting opportunity in a growing company with a solid growth path and mentorship from a strong group of senior developers.


Responsibilities:


*	Direct the execution of account management requests. 
*	Ownership of account management system. Working within an existing code base, extending it, and developing new applications with it.

·          Ownership and upkeep of code base repository 

*	Develop functionality enhancements to existing products
*	Manage back end systems


Technical Qualifications:


*	Mastery: Object Oriented PHP, MySQL ( design , usage and reporting) 
*	Familiarity: MySQL query optimization, experience with Java , Code Management and version control, Linux command line, INNO DB engine, payment gateways such as payflow.
*	PLUS: Perl, Ruby, XML/XSLT, experience with database reporting systems.

Professional Qualifications 

 

*	Candidate should exhibit great attention to detail and be able to work effectively on multiple projects at once

*	Candidate should have strong interpersonal, organizational, and communication skills 
*	Highly motivated, self-directed, deadline-responsive and able to work effectively under pressure
*	Excellent project management skills and ability to develop and maintain deadlines
*	Possess Vision and Adaptability 

This is a full time, on site position. We offer competitive compensation including Vacation and Holiday Pay, Medical/Dental and Life.
