CBS Local is looking for a strong PHP developer to join our development team. The senior developer will work on new features and products related to the CBS Local Digital Media websites. CBS Local is consistently in the top 5 local sites as measured by Comscore and we're working on new web and mobile products to support additional growth. We provide a great work environment where developers can learn and grow their skills and have the opportunity to work on new products and technologies.

Candidates must have experience in developing web applications using WordPress, PHP and MySQL.  Additional skills in HTML, AJAX, JavaScript and CSS are also desired. Any mobile or location based development is a plus.

Responsibilities:
 
Analyze, design and develop websites using PHP, HTML, JavaScript, CSS and MySQL.
Work with the development team and product managers to customize our WordPress CMS.
Build new features into the sites using third party APIs such as Facebook and Twitter.
Perform code reviews for junior developers.
Work with the product team to help define technical requirements and capabilities.
 
Required Skills/Experience	

Proven 8 to 10 years experience in developing high availability, high volume web based applications using PHP, MySQL and WordPress.
Experience developing, debugging, and performance tuning WordPress plug-ins and themes.
Strong HTML, JavaScript, and CSS development skills.
Experience with implementing APIs from social networks like Facebook and Twitter.
Strong written and verbal communication skills.
Excels at working in a highly collaborative environment.

Please send your resume to jobs at cbslocal.com
