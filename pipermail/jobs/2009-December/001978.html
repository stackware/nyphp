Front-end Developer
ChallengePost is a fast-growing and funded NYC-based web startup. It's  
where individuals or organizations go when they want something that  
doesn't exist, so they can challenge the world to make it or do it. We  
have big ambitions. We value those who can execute and self-manage,  
and we like creativity and passion for delivering a great user  
experience.

Front-end developers are responsible for implementing, testing,  
debugging, and troubleshooting clean, web standards-compliant code.  
They will work with our server-side developers to integrate their  
templates and client-side scripts into the final product.

2-3+ years experience in front-end web development
Mastery level knowledge of HTML and CSS is a must
Expert knowledge of JavaScript
Experience translating comps and wireframes into flexible and reusable  
code templates
Strong working knowledge of developing cross-platform and browser- 
compatible code (IE, Firefox, Safari, Opera, etc.) for dynamic web  
applications
Familiarity with and adherence to web standards and best practices for  
creating ARIAs
Experience with version control software is a plus
Effective verbal and written communication skills
Strong analytic and problem solving skills
Basic familiarity with JavaScript frameworks and animation libraries
Expert knowledge of DOM manipulation
Familiarity with PHP frameworks (Zend, Symfony, CakePHP is a plus)
jQuery/AJAX and Object Oriented JavaScript experience a must
Please email resume, cover sheet, and code samples to brandon at challengepost.com
