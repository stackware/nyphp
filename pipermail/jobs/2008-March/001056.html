About Oddcast:

*	Design and develop database infrastructure and front end UI 
*	Code, test and debug applications from written and verbal

*	Utilize PHP, mySQL, MS-SQL, ODBC, XML, JavaScript, DHTML 

Requirements Include:

*	Experienced in development of rich media & eCommerce web
