Description:

Our client, one of the leaders in the real estate industry, is seeking a
talented and dedicated PHP Developer with a proven track record . The ideal
candidate will have at least 5 years of professional experience building
commercial, enterprise PHP and MySQL web applications. This is an on-site
position, based in our midtown Manhattan office. Only local applicants who
are authorized to work in the US will be considered.

 

*	Responsibilities 
*	Support existing web software applications and design and develop
new applications within the legacy system. 
*	Deliver projects built to scope, according to agreed upon goals 
*	Create and perform unit and integration tests 
*	Create and maintain code documentation 
*	Debug and troubleshoot software defects


Qualifications

Applicants should possess the following qualifications:

*	5+ years professional experience in object oriented PHP5 and MySQL
application development. 
*	BS in computer science or related discipline, more advanced
technical degree preferred. 
*	Expertise building applications using PHP5, MySQL, SQL and AJAX.
MVC is preferred. 
*	Must be able to develop reusable PHP application components 
*	Must possess experience in working with various frameworks such as
Zend, database abstraction frameworks and Perl modules. 
*	Solid understanding of object oriented programming concepts and
principles. 
*	Significant experience building Zend Framework applications 
*	In-depth knowledge of software engineering processes and best coding
practices 
*	Ability to learn quickly and take ownership of tasks, process, and
products 
*	Excellent communication (written and verbal), time management,
troubleshooting, analytical, and interpersonal skills 
*	Ability to work independently and as a team member in a dynamic work
environment 
*	In depth knowledge of HTML, CSS, JavaScript, and Templating 
*	Experience in working with Linux Servers (not just in a hosted-site
environment) is desirable.
