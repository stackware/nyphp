*Job Summary*

We're looking for a skilled PHP developer who possesses an in-depth
knowledge of web application development.  This position requires a mastery
of the hands-on aspects of front-end web interface engineering, an ability
to design and build solid architecture that integrates with other systems
and technologies, and the ability to communicate these solutions to other
members of the team both verbally and in written documentation.  The ideal
candidate will have extensive experience authoring object-oriented PHP5
applications with an MVC framework, but will be excited to dive into other
technologies should the need arise.



Overall day-to-day responsibilities include leading front-end engineering;
the hand coding of web-based applications, web pages, emails, and web-based
mobile experiences; integrating your code with other technologies (REST &
SOAP web services, clients’ back-end systems, content management systems,
etc); performance optimization; and working with visual designers,
interaction designers, and software developers.



Responsibilities:

·       Developing and testing PHP/MySQL, XHTML, CSS, and JavaScript based
web applications

·       Consulting with creative team during wireframing/technical
specifications phase, helping to vet projects and suggest new technologies

·       Working with the front-end development team to create seamless user
experiences

·       Working with the back-end development team to integrate 3rd party
content management systems, e-commerce platforms, and other data sources

·       Contributing own sense of visual integrity, animation timing, flow
and UI planning to all projects



Required Skills and Experience:

·       Minimum 3+ years experience utilizing front-end technologies
including XHTML, CSS, JavaScript

·       Minimum 3+ years experience building object-oriented PHP/MySQL web
applications, preferably using a MVC framework such as the Zend Framework

·       Extensive experience in enterprise application deployment
architectures, environments and concerns (scalability, performance,
availability, reliability, security etc.)

·       Working knowledge of Linux, Apache and MySQL Server Administration.

·       Experience modifying and extending open-source software such as
content management systems and e-commerce platforms

·       Experience with distributed source control (Git, Mercurial)

·       Solid understanding of cross-browser/platform issues and solutions

·       Exceptional time management (can meet deadlines), problem solving,
teamwork, and communication skills

·       Ability to follow technical specifications and production processes

·       Ability to write well-commented code and documentation

·       Must be available to work on site, but will have future possibility
for remote work



Skills and Experiences That Are a Plus:

·       jQuery, Prototype/Scriptaculous, Ext JS

·       Zend Framework (or other MVC application framework)

·       Building Magento stores: developing custom themes, developing custom
modules, implementing 3rd party extensions

·       Writing unit and functional tests as an integral part of your
development process

·       .NET Framework, SQL Server

·       Enterprise technologies such as Web Services and MS SharePoint

·       Integrating major 3rd party APIs (Facebook, Open Graph, Twitter,
Google Maps, YouTube, Open Social, etc…)

·       Previous interactive marketing agency experience - working with

·       Musical or other creative talents

·       Proficiency in Microsoft Office, Photoshop, and presentation

Education and Experience

·       Undergraduate degree in Computer Science, Software Engineering or
