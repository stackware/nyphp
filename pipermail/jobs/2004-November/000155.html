Subject: [YNPN-NYC] PHP/SQL/Web developer (FT): The Open Planning
Project
Company Description
The Open Planning Project (TOPP) is a nonprofit organization dedicated
to developing tools to enhance the role of technology in urban planning.


Job Qualifications
You must have:
+ Bachelors Degree or above from a top school.
+ Very strong programming experience.
+ Experience developing PHP and SQL driven websites.
+ Good knowledge of HTML, DHTML, Javascript, and CSS.
+ Ability to code without the use of a WYSIWYG editor.

For a Perfect Fit:
+ Experience with Wikis or MediaWiki is a tremendous plus.
+ Experience/Interest in community organizing and digital community
  building.
+ Knowledge of Perl, Java, c/c++
+ Illustrator and graphic artist experience, fantastic. Experience
+ working on Linux servers with apache via ftp, sftp, also good.

Please e-mail resume to: jarasi at openplans.org

Job Description
The Open Planning Project (TOPP) is looking for a web developer with
strong programming and design skills to be responsible for all aspects
of concept, design, and development for a new project.  We will be
building a community organizing and information sharing tool based on
wiki concepts and technology.  You will be responsible for adding new
functionality to existing open source code to produce a powerful new
tool to enable communication for public interest groups.
