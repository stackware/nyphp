PHP/MySQL Consultant | 50-65/hour - NYC

MUST WORK ON-SITE IN NEW YORK CITY - NO TELECOMMUTERS 
YOU MAY BE ABLE TO WORK FROM HOME, BUT ONLY AFTER A FEW FACE-2-FACE
SESSIONS 

The PHP/MySQL Consultant will code web applications using PHP/MySQL.
This role will take complete ownership of the project: anticipating
challenges, providing technical recommendations, and creating work that
is stable, usable, and scalable.

RESPONSIBILITIES: 
* Will manage existing applications and help create new applications
from the ground up 
* Need to be extremely detail-oriented and meticulous but able to
quickly assess situations and adapt 
* Should code HTML, CSS, Javascript, and PHP / MySQL. Should be able to
vigilantly identify and remediate unvalidated inputs and inefficient
code 
* Should troubleshoot problems quickly and anticipate future ones 
* Most important, need to be a leader: The client needs someone who can
take ownership of a project, quickly identify points of pain, determine
action steps and priorities, and code efficient solutions 

QUALIFICATIONS: 
1. At least 4 years experience with PHP/MySQL, Javascript, XHTML, CSS,
and XML 
2. Experience with enterprise-grade web applications 
3. Good user experience and usability intuition 

If you are interested in this position and live in the NY/NJ area,
please submit your resume & hourly rate to beau at open-source-staffing.com
