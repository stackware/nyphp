PHP Developer (with Release Management Experience)

Key Skills: 
LAMP, SVN, Release Management

Job Description:
Guidepoint Global is seeking a PHP Developer with Release Management
Experience  to join its top-flight IT team. Our web development team is a
highly collaborative organization consisting of engineers with the highest
levels of technical depth, programming skill and a passion for quality. 

 

This position is expected to write cutting edge web application using highly
object oriented PHP 5 code and collaborate with project management and
quality assurance staff on the release of multiple web projects.

 

About Guidepoint Global:
Guidepoint Global is an independent research firm based in NYC that provides
customized primary research and consulting services to premier institutional
investment firms that invest in the Healthcare, Technology, Telecom, Media,
Retail, Energy and Industrials industries. We provide our clients with
customized tools to analyze various sectors and sub-sectors via the
knowledge and insight of industry professionals that make up our proprietary
Advisor network.

Duties and Responsibilities:

-          Design, develop and test applications using PHP.

-          Setup the necessary process and tools for effective release
management. 

-          Coordinate the software releases, build, package and deploy to
the QA, staging, and production environments.

Requirements:

-          LAMP (3+ years)

-          Expertise in Configuration Management (Change Management best
practices and automation, Source Code Control, Deployments, Builds)

-          Strong Linux skills (command line, scripting, administration)

-          Experience working with a production environment with zero or
very minimal downtime

-          Strong work ethic plus ability to multi-task

-          Excellent communication skills

Strongly Preferred:

-          B.S. Computer Science or related degree

-          Experience with unit testing frameworks i.e. PHPUnit

-          Experience with System Testing i.e. Selenium
