I have a client with a need for a PHP API Developer.  It is a 3 month contract in the Fairfield CT area.  This is an on site positin, no remote work available.  I'm including the job description.  If you are a fit please forward your resume and rate with contact information.

GENERAL RESPONSIBILITIES        

We are seeking a candidate for a 3-month consulting assignment. The candidate is responsible for developing the PHP version of client's proprietary AAA (authentication, authorization and auditing) API. 

The candidate is expected to report to follow the provided functional spec and design document to roll out the API implementation, conduct effective unit test, provide technical support to QE and other teams of the project, support integration with other modules and conduct a thorough knowledge transfer at the end of the assignment. 

The candidate is also expected to write/update functional and design spec documents and come up technical solutions to address client's digital certificate management need.

REQUIRED EXPERIENCE/SKILLS:

·       5+ year experience of working as a senior software engineer; must possess solid knowledge of SDLC, hands-on experience of using UML in software design and implementation, and hands-on experience in unit testing.

·       Must have the ability to digest technical spec and design documents and follow them to roll out software implementation with minimal support. Equally important, must have the ability to troubleshoot and come up with technical solutions for challenges encountered during the software implementation.

·       Must have excellent communication skill to write software functional and design specs.

·       Hands-on working experience of using PHP LDAP API to access LDAP database (experience of OpenLDAP database is highly preferred).

·       Hands-on working experience of Kerberos technology (e.g. MIT Kerberos 1.6.3) is highly preferred; experience in using C++ Kerberos API is highly desired.

·       Hands-on working experience of using PHP API to access MySQL database is required (experience of managing MySQL user accounts through API is highly desired).

·       Hands-on experience of designing, writing and executing effective unit test for PHP (e.g. PHP unit) applications is highly desired.

·       Hands-on working experience of Java programming and JNI is highly desired. 

·       Experience with ClearCase is preferred.


REQUIRED EDUCATION

·       Bachelor Degree in Computer Science or Electrical Engineering is required.

·       Master Degree in Computer Science or Electrical Engineering is preferred.
