Hey Sharkbait, perhaps I could save you some cash-

Assuming you are trying to make a web site out of the documents, this would be really easy to automate using the wiki function of the Vork PHP framework (http://www.Vork.us) which lets you paste Word documents into a WYSIWYG textbox and it will convert it to XHTML-valid markup before creating your wiki page. Just install Vork, create a new controller and set it to be a wiki (just add to an action- get::component('wiki')->wiki();   ) and then navigate to the page and use the wiki interface to add a new wiki page for each Word document.
