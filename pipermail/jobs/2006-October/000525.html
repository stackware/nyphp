Network, is one of those participants).

Right Media has built the online advertising industry¹s first open media

The exchange and our media network (one of its thousands of participants)

---Lead/Senior PHP Developer---

The core technology is developed in a C/Unix environment. The UI and
reporting tool is developed in PHP. Due to growth and increased development
needs, we are seeking experienced PHP Developer to join our user
interface/reporting team.

Primary Responsibilities include:

All candidates MUST have significant professional experience developing
complex, database-driven PHP applications. This is a challenging position
and is not suitable for novice or entry-level programmers. As the
application uses a database backend and runs on Unix, experience with SQL
(MySQL experience is desirable) and Unix/Linux is required.
The developer will be responsible for the creation of a web application that
displays complex, custom reports. The developer must have strong
intrapersonal skills and be able to find the ideal way to implement
requirements in a clean, elegant way. Reusable code is important, using
object-oriented design where possible. Functional tests will be written,
where possible, to validate the correctness of the code.
Candidates should be able to provide examples of web applications and/or
frameworks that they have developed, to discuss practical and theoretical
consequences of design choices.

Primary Qualifications include:

-  Professional experience developing/architecting PHP or Java-based based
web applications
- SQL and a good knowledge of client-server architecture a must
- Web page design with DHTML, AJAX, and CSS
- Experience administrating Apache/PHP/MySQL in a Unix environment is a plus
- Proven experience designing browser-based GUIs, writing usability tests,
and incorporating user feedback


---Quality Assurance Engineer---

Right Media is currently seeking a Senior Quality Assurance Engineer to join
the development team. This position will coordinate and conduct QA testing
activities, ensuring that the new software version meets stated functional
requirements and runs error free before it is released for live operation.
Additional responsibilities will include creation of on-line and printed
technical material to be shared with other development teams across the
company intranet.

Primary Responsibilities include:

€ Manage small testing team
€ Work within the established QA process, including test planning, test
design, and test execution/reporting
€ Coordinate and assist in developing an inventory of test plans, standard
test scenarios, detailed test scripts, test verification procedures, and
associated test case data for all NY supported functions
€ Plan, coordinate and take a major role in conducting the following types
of QA testing
€ Assist in the Unit/string testing of all pre-production web site
enhancements released to the test environment by the development staff
€ Beta testing of new releases
€ Integration testing of new software releases and other necessary
interfaces
€ System/regression testing to ensure overall application integrity of new
releases
€ Acceptance testing to gain user approval of new releases
€ Any additional testing deemed necessary by Management (e.g., volume/stress
testing, security testing, business continuity/contingency testing, selected
pilot/parallel testing)
€ Investigate and log test discrepancies and bugs. Work closely with project
manager and developers to assist in communicating testing results and in
resolving test discrepancies
€ Assist development management in planning new releases

Primary Qualifications include:

€ BA/BS in Computer Science, related degree, or comparable experience
€ 5+ years of professional software testing experience
€ Broad experience in writing and executing test plans and test designs.
€ Solid background in both application systems and business processes
€ Strong problem solving and analytical skills
€ Proven ability to both work independently and in a project environment
€ Effective team player
€ Strong attention to detail and accuracy
€ Strong written and oral communication skills
