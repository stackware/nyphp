NYTimes.com is seeking the following:  Expert PHP-MySQL-Linux Developer
We are seeking a candidate who is confident about building and extending
the backend of a personalization platform for nytimes.com, a site with an
unparalleled combination of scale and complexity. You have extensive
experience programming in a UNIX/Linux environment. You are a self starter
who will pick up programming tasks and run with them, collaborating in a
team environment with buy-in, checkpoints and testing.

Responsibilities include the following:
   Developing key PHP classes and components for the server side of an AJAX
      platform
   Considering client, protocol, server, performance and security issues
      during development
   Contributing creative ideas toward solving platform issues as they crop
      up
   Designing reusable, object-oriented code that can be redeployed
   Ensuring robustness and performance goals through unit and performance
      testing

Necessary skills include the following:
   B.S./B.A. in Computer Science, or a degree in a related field plus
      Computer Science coursework, or equivalent experience
   Over 6 years Web development work experience
   At least 4 years PHP 4-5 experience in an Apache, Linux, MySQL
      environment
   Strong knowledge of MySQL, performance issues, and feature trade-offs
   Experience with high traffic websites
   In-depth knowledge of HTML, CSS and the major web browser platforms
   Object-oriented design in PHP
   Experience with XML, XML-RPC, XSLT and JSON
   Text parsing, regular expressions, validation rules, and security.
   Experience working as part of a team
   Strong verbal and written communication skills

Desired skills include the following:
   Formal software development methodology
   Experience writing shared and reusable components of code
   Experience designing and tuning applications that can scale
   Some background in another structured programming language, such as Java
      or C/C++
   Use of source control (CVS, SVN)
   UNIX/Linux: Proficiency as a system user and light administration
      skills.
