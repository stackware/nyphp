FRONT END WEB DEVELOPER - (HTML/CSS Coder)
Matthew Schwartz Design Studio (www.ms-ds.com), an established,  
growing, interactive and print design studio seeks a top-notch front- 
end code designer & developer to contribute heavily to both our  
design and development teams. The ideal person understands the  
importance of working in a deadline-driven environment, is fluent in  
core front-end technologies (HTML, CSS, XHTML, JavaScript, Ajax,  
Flash), has an intuitive grasp of UI design, is highly creative and  
understands what “done” means.

We’re a professional but very easy-going group. We'd like someone who  
plays well with others (particularly graphic designers and  
programmers) and has a sense of humor.

PRIMARY RESPONSIBILITIES: Develop, write, test and debug front-end  
code to implement the UI design and functional specifications for  
websites, interactive ads and web applications
*Translate design across site experiences and create templates in  
CSS, XHTML
*Be able to modify PHP source to add XHTML templates
*Add UI detailing and improvements with JavaScript and AJAX
*Flash and ActionScript 2.0 design and development
*Produces documentation for the use of other Web Developers
*Writes clear, maintainable, portable code

REQUIREMENTS:
*2+ years of professional web development experience
*Strong working knowledge of Java Script; CSS; XHTML; AJAX
*Decent working PHP knowledge (need to be able to integrate CSS and  
XHTML into PHP code)
*Solid experience with Flash
*Strong knowledge of web design principles and technologies
*Experience working in LINUX, MAC and Windows environments
*Basic understanding of Web software applications used in web design  
and development and is recognized as a resource in those areas
*Experience deploying across multiple platforms and browsers desired

BENEFITS:
--Excellent benefits: healthcare, paid vacation time, summer flex  
hours, 3% company match for retirement plan, TransitCheck, company  
match for professional development fees
--Great working environment, established and stable, creative & not  
corporate
--Room to grow

COMPENSATION:
--Competitive base salary plus additional bonus incentives
