You may want / need to create a new User Group, and assign it custom
permissions - something that only allows access to the components you want,
while (by default) not allowing access to everything else.  TEST THIS
THOROUGHLY - remember that a single "Deny" will override any and all
"Allow" and "Inherited" options when it comes to permissions.
