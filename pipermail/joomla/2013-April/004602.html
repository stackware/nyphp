Not sure if I'd say it is the "Best" but I do have a beta of Raptor 
working on Joomla 3.0 and hopefully can polish it up and release it soon.

It's more the "funnest" editor I've used in a while, because it gets rid 
of the clunky little menu bar on every edit box and instead moves all 
that functionality up to the top of the screen.  Whats really nice is 
that I find you can really "see" what your edits will look like using it 
because you can edit things in place.
