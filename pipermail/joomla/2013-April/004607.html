I don't know of any tutorials, but that would be a great idea!

Depending on what you want changed, some of it can be done with CSS. Select Properties and there is an HTML layout to change that part of it. You can put in CSS internal for the form, but most favor to put their CSS in an external file.

Their technical support has been good, ask them questions if you get stuck with things.
