Ooops. My bad. I didn't read the whole post before I responded.

I did see the "View All Categories" menu mid-way down on the left side of
the page.

I have never a vertical menu like this built into a Joomla theme. It seems
like every fly-out menu is horizontal.

You might be better off sticking with Yootheme and making your own menu
using jQuery - you can find lots of neat vertical jQuery menus on the web
at places like DynamicDrive.com

Also, looks like a neat idea for a Joomla module - a fly away vertical menu.
