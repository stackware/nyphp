Geoffrey, you said it right! The possibility of using CSS styles within the
editor window is fantastic!!!

Marcos, you can configure JCE to see your template.css file, though I don't
find this the best choice.

You REALLY need to know your CSS and how it works. Together with Firebug, a
Firefox add-on, you find the IDs and classes used by JCE within the editor
window, add them to your template.css file and BANG! True WYSIWYG in
Joomla!!
