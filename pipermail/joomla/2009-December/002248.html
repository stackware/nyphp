Now, due to my own ignorance at the time, I disabled ajax and rewrote the
search pages a bit.........   but now that I'm familiar with it, I could do
the same and keep ajax on.

What I did find with the templates is that there are a lot of formatting
instructions that are applied to the default variables - ie they are already
html coded so it can be difficult to do some things.  But you can get access
in the template to the raw data if you need it[for example, I needed raw
data in order to create the google static maps] and then you have a lot more
control.
