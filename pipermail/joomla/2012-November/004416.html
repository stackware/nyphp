I got this figured out - it was a combination of server time, and the time
setting within Joomla.

Originally, both the Server and Joomla were set to America/New_York - I
embedded a time function into a module, and saw PHP return a value that was
the current time, but in GMT instead of EST.  Once I changed Joomla to be
GMT, it returned the correct time, and the article published normally.
