If you redirect www.domain.com/administrator to
www.domain.com<http://www.domain.com/administrator>,
no one will ever be able to log into the back end ;-)

My recommendation is to use Admin Tools,
https://www.akeebabackup.com/products/admin-tools.html, which allows you to
require a secret key to access the admin login.
