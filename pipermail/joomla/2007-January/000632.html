Thanks for the suggestions by everyone. I installed OPENSEF and it 
solved the problem. It does allow you to take a menu item, such as the 
FAQ example and be able to have it referred to externally by whatever 
URL you wish such as /faq/my-faq.html. It also does some intelligent 
naming based on the title for other pages URLs as well, but you can be 
specific of what you want for each URL. The elegant part of this is 
that it doesn't make the user aware that a redirect (even if it is) is 
taking place.
