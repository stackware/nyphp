Ha 50 minutes, could have done it faster but I got fancy and added both menu ids and content ids

https://github.com/AmmoniteNetworks/Joomla-Content-Access

It checks if the current MENU id is active and if so it replaces all the content items text with the contents of the default.php  in the plugin folder.

if it is the content ID it only does the  content item with that ID.
