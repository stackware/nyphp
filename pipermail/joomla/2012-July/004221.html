I am trying to use captcha-recaptcha on a simple joomla contact form.
But the captcha code is not showing up.
Here are my settings:
The Administration Control Panel (ACP) has the following Captcha settings in various locations:


Global Configuration: ACP > Site > Global Configuration > Site tab > Default 
Captcha: Here there are 2 different options: (1) “- None Selected -” and (2) “Captcha – Recaptcha”

Contacts: ACP > Components > Contacts > Options > Form > Allow Captcha on 
Contact. Here there are 3 different options: (1) “- Use Default -“, (2) 
“- None Selected -“ and (3) “Captcha – Recaptcha”.

Plug In: ACP > Extensions > Plug in Manager > Captcha – Recaptcha plug 
in: Here one needs to enable the plug in and insert their (1) Public Key and (2) Private Key obtained by registering at http://google.com/recaptcha.


User Manager: ACP > Users > User Manager > Options > Component > Captcha: 
Here there are 3 different options: (1) “- Use Default -“ (2) “- None 
Selected -“ and (3) “Captcha – Recaptcha”
Currently, I have the following settings applied:


Global Configuration: Captcha – Recaptcha
Contacts: Use Default
Plug In: Public and Private Key entered correctly
User Manager: Use Default
I have also tried the following settings:


Global Configuration: Captcha – Recaptcha
Contacts: Captcha - Recaptcha
Plug In: Public and Private Key entered correctly
User Manager: Captcha - Recaptcha



I have a global key from google  (just acquired it, has not be used on a site before)
