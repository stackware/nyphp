Mosets Newsletter: Introducing Mosets Tree 3. Now with multi-directory support.  
 
  
 
Introducing Mosets Tree 3.
Now with multi-directory support.
 I'm pleased announce the release of Mosets Tree 3.0, a major update to to one of the most popular directory extension for Joomla.
Mosets Tree now features multi-directory support that allows you to create any number of directories with a single Joomla site. A new innovative feature called 'Category Association' opens up a completely new way for you to organize listings you've never been able to previously.
The new Mosets Tree also improves the way your users browse and search for the listings they are looking for, quickly, with the new "All Listings" page and Filter search.
Learn More View Demo  
What's new at a glance
	* Multi Directory
	* Category Association
	* User Profiles
	* Module Assignment Options
	* New Field Types
	* Captcha
	* Server Side Form Validation
	* All Listings
	* Filter Search
	* New Menu Item Types
	* Link ID & Alias Hybrid SEF URLs
	* Open Graph
	* Custom Categories Ordering
	* and more! 
Get it today
Mosets Tree 3.0 is available for download to all active subscribers by logging in to their Mosets account. If you are not a subscriber yet, you can purchase a Mosets Tree subscription to become a Mosets Tree subscriber today.
I hope you like it.
