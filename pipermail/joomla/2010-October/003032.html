Paul,
Are they the same version of Joomla?
Do a fresh install with sample data. This way you know that install works.
Install the template you want.
Download the new configuration.php
Create a new empty DB
Upload the old db to this new DB
Change the configuration.php to match this new DB
That should work.
If not you have other issues. What type of hosting do you have?
If you look at the old db is the data there?
