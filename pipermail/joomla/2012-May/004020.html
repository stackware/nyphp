Hi Janet!

*index.php* should be under your folder *templates/tk_redemption_free*

On the header are you should find a line like this:

<link rel="stylesheet" href="/templates/*tk_redemption_free*/css/custom.css"
type="text/css" />

If it's not there, you can create it.

The *custom.css* file should be under *templates/tk_redemption_free/css*

If it's not there, you can also create it.
