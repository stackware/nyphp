When looking at a web site through the web browser, do a view source
and see if there are generator tags or the like that give a clue to
what CMS it might be. I know with Joomla, if looking at a Joomla web
site this information is available. Unless it was removed for security
purposes.

When I've been curious about a web page, I've e-mailed the webmaster
to ask and often get a helpful response.
