Paul,

Is this still a problem? Are you running PHP on windows or on linux?

If you are running it on windows, you should really use the Sql Server
Driver for PHP  http://www.microsoft.com/download/en/details.aspx?id=20098.

If you are running PHP on ubuntu, the mssql driver is not available from
the default repositories. I think you will have to build it yourself.
