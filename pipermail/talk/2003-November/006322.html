Hi Folks:

I've whipped up a patch file in Unifed Diff format which I apply to my new 
PHP 5 installations.  Do note, this is a _development_ environment, so not 
all settings are appropriate for a live deployment.  But, this patch file 
approach is helpful tool for upgrades, regardless of what settings you 
use.

Enjoy!

--Dan


vvvvvvvvvvvvvvvvv ini5.sh vvvvvvvvvvvvvvvvvv
#! /usr/bin/sh

patch -i f:/php/ini5.diff c:/progra~1/php/php.ini-recommended -o c:/progra~1/php/php.ini
chmod 755 c:/progra~1/php/php.ini
