For a quick comparision of what's going on between the servers, do a 
phpinfo() call inside a script or execute "php -i > phpinfo.htm" on each 
of the servers and compare the difference.

I suspect the new server might have safe_mode on or might be an apache 
module as opposed to CGI.

On the new server, you might be able to use their CGI with your php.ini 
file or build your own php for CGI use.
