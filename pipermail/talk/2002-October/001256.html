try eval

<?php
$string = 'cup';
$name = 'coffee';
$one = 'This is a $string with my $name in it.<br>';
echo $one;
eval ("\\$two = \\"$one\\";");
echo $two;
?>

will print

This is a $string with my $name in it.
This is a cup with my coffee in it.
