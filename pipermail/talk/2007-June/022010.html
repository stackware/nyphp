No javascript required, just have the client use Safari. (just kidding)

As others pointed out, hiding the status bar is considered a security
no-no because it is often the only way for a user to discover
cross-site scripting and phishing attacks.

Even if you can control it in older browsers, you probably won't be
able to in newer ones.
