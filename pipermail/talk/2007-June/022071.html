Hello, 

I believe you need to set the following:

    $emailObject   = new PHPMailer( );
    $emailObject->SetLanguage('en', "${OOP_INC_PATH}/language/");

Where $OOP_INC_PATH is the path to your PHPMAILER directory and the language directory.
