Thanks for the feedback

The error was super simple:

        $xml = new DOMDocument;
        $xml->loadXML($this->_payload);

        $xsl = new DOMDocument;
        $xsl->load($this->_filePath);

        $proc = new XSLTProcessor;
        $proc->importStyleSheet($xsl);
       
        return $proc->transformToXML($xml);    

I was calling load(STRING) vs loadXML(STRING), so when it attempting to 
transform the XML, it (the dom doc) was empty - it didn't load the xml 
string data.
Too bad it doesn't throw an error or exception if an argument is incorrect.
