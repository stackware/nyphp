As it turns out, the latest combination of PHP and Apache that works is: 

Apache 2.2.3 and PHP 5.2


If one wants to use any Apache version higher than Apache 2.0.x 
(like I was using Apache 2.2.3) then one must download the most recent 
snapshot of PHP, that is PHP 5.2 available at this php.net sanpshots 
downloads page: 

http://snaps.php.net/

I've done the above, updated my Apache .conf to point to the newer, 
correct PHP 5.2 .dll called: php5apache2_2.dll

Already up and running. Thank you all.

Peter
