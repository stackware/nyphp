Why not just use the built-in function http_build_query? 
-------
The easiest way I've found to do this is as follows:

$myvars = array(
  'item1' => 'value1',
  'item2' => 'value2',
);

$postdata = array();
foreach ($myvars as $k => $v) {
  $postdata[] = urlencode($k) .'=' . urlencode($v);
}
$postdata = implode('&',$postdata);
