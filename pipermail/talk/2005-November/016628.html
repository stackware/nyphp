This is usually solved through the most famous process LMAYAFSQ: Let
me ask you a few stupid questions:

- Did you run setup on your new installation?
- If previous failed, have you tried to install the new version on a
new folder and run setup on it?
- Have you tried the second one, had success, exported all the data on
previous mysql version and imported it to the new version? (this is
what I'd do myself and it always works)
