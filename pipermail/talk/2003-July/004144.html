>> Are you trying to log that session ID if using cookie-based sessions?

>> Here's an example from my httpd.conf file for the log format that

>>         LogFormat "%h %l %u %t \"%r\" %>s %b \"%{Referer}i\"

>> ...which simply adds \"%{Cookie}i\" to log the stock combined log 
>> format.
>> You'll get all cookies stored by your site, but at least you have the 
>> session
>> ID there for use.
