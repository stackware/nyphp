> Ampersands refer to references.  Example:
>
> <?php
> $x['a'] = 'one';
> $x['b'] =& $x['a'];
> $x['c'] = 'three';
>
> var_dump($x);
>
> $x['a'] = 'two';
>
> var_dump($x);
> ?>
>
> Output:
>
> array(3) {
>   ["a"]=>
>   &string(3) "one"
>   ["b"]=>
>   &string(3) "one"
>   ["c"]=>
>   string(5) "three"
> }
> array(3) {
>   ["a"]=>
>   &string(3) "two"
>   ["b"]=>
>   &string(3) "two"
>   ["c"]=>
>   string(5) "three"
> }
>
> Notice how I changed the value of b by changing the value of a.
