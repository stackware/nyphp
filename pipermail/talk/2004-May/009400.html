I would tell him to check that apache is set up properly.  Make sure 
that his httpd.conf file has this line:
AddType application/x-httpd-php .php
If apache isn't told to pass the file to PHP it will just spit it out as 
is and the users browser will ignore any tags (<?php ?>) it doesn't 
recognize.
