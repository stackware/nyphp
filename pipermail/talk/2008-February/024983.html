Description:

The Semantic MediaWiki (SMW) allows users to add structured data to
MediaWiki pages through simple wikitext markup that identifies
relations between pages and attribute values of pages. With this
information, SMW can help to search, organize, browse, evaluate, and
share the wiki's content.

Semantic MediaWiki
http://semantic-mediawiki.org/

Semantic Forms are an extension, based around Semantic MediaWiki, that
allows users to create forms for adding and editing pages that use
templates to store semantic data. Forms are defined using editable
text files, written in a custom markup language, that are then parsed
on-the-fly when a form is needed.

Semantic Forms
http://www.mediawiki.org/wiki/Extension:Semantic_Forms

MediaWiki is a free software wiki package originally written for
Wikipedia. It is now used by several other projects of the non-profit
Wikimedia Foundation and by many other wikis.

MediaWiki
http://www.mediawiki.org/
