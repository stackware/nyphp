In FF address bar, type "about:config" to access the configuration
area.  Right click anywhere, choose "new" and add a "String".

Enter the Preference Name to be "general.useragent.override" and use
whatever value you wish.

Also, I found this: http://extensionroom.mozdev.org/more-info/useragentswitcher 
not sure how good it is though.
