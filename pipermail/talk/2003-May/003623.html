Meanwhile, your $query is a mySQL reference pointer, not an array.  To
obtain an array out of the resultset pointer there are several ways in PHP
to do it, one way that pops up in my head (probably not the best way but..)
is mysql_fetch_rows:

for ($i = 0; $row = mysql_fetch_row($query); $i++) {
 echo "<option value='$row[0]'>$row[0]</option>";
}
