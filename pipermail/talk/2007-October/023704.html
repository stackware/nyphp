The simplest way I can think of is:

if($Cost < 100)
{
   include('script-a.php');
   exit();
}
else if($Cost > 1000)
{
   include('script-b.php');
   exit();
}
// if the above two "exit's" didn't get invoked, rest of the script will be
processed.
