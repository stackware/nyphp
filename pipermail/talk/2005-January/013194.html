SHOW is a keyword in 4.0.  Try using backticks so mysql knows you're 
defining a column name and not a SHOW statement:
...  pic varchar(50) default NULL,
  `show` tinyint(1) unsigned NOT NULL default '1', PRIMARY KEY  (s_id) ...

- Allen
