In light of some recent discussion about a Python user group in NY, I
talked with some folks about getting something going for Python
programmers.

Since Python is part of the LAMP stack, we've created a Python SIG at
New York PHP.

http://nyphp.org/content/mailinglist/mlist.php

http://lists.nyphp.org/mailman/listinfo/python


It's not a full blown user group, but it's a start.  Anyone is of course
welcome to join and discuss.

Also, anyone is welcome to contact me to get involved in organizing the
SIG and events.  New York PHP will help however it can with resources,
servers, etc.
