Heya:

> If by "do this" you mean to see if the array changed, what about  
> array_diff()?

And be aware that the order of arguments impacts the results.

$a = array(
    'a' => 'one',
    'b' => 'two',
    'd' => 'four',
);

$b = array(
    'a' => 'one',
    'b' => 'two',
    'c' => 'three',
);

print_r(array_diff($a, $b));
print_r(array_diff($b, $a));
