> kept the belief that RedHat is better for servers while Ubuntu is preferred

I prefer a distribution that is as stripped down as possible, so that I add
only the things that need to be there. Any distro that installs an X server
by default is out.

But really, choice of distro is a personal preference, like choice of text
editor. They all do the same thing, but in subtly different ways. Go with
what the sysadmins will be happiest working on.
