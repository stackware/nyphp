On Fri, Sep 2, 2011 at 12:42 PM, Rolan Yang <rolan at omnistep.com> wrote:
> What's your thought on using XAMPP instead of creating a bunch of virtual
> machines?

XAMPP isn't ideal because it doesn't give you a server environment.
There are a lot of subtle (and not so subtle) things that change when
you move from one environment to another.

Developing against a virtual machine with the same configuration as
your production server gives you a lot more confidence that your code
will work as intended in production.
