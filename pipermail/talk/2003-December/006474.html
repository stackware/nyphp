I was wondering how to do this a few days ago. I found this little program called swiftsurf which is a basic http proxy. Its very easy to setup. As long as one does not use add blocking it works fast. One just needs to set  "request 1   answer 1" in the config file to watch all the headers. I like this solution more because it lets me spy on any type of browser i use.

On a related note. I generaly test with Firebird but I took a look at IE and it seems to not want to send If-Modified-Since header even when in cache settings i checked of to check on every page access. Or this could be that IE with wine just dont run right ?
