It is going to be really hard to beat Shopify.  You can get a store up
and running in no time, and it is very cheap.  If you have less than
100 SKUs it is only $30/month.  A VPS that can handle Magento will
cost about that much.  Do not try and run magento from a shared
hosting account, Magento is a massive PHP application that can be
painfully slow.
