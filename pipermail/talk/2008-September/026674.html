Hi Mike,
   It kind of sounds like you are talking about two different things
or at least the confusion involves two different topics. Basically, if
you are going to use a Switch statement it's a good practice to keep
the case expression within quotation, if you want to get desired
results.

 Also, be very careful when doing ,
SWITCH($foo)
{
case  ' 5 ' :
case ' 6 ' :
    do execute some code...
break;
default:
break;
}

also 5 is an interger, but ' 5 ' is a string. So that will make a
difference when comparing it to $foo. Take a look at
http://us.php.net/intval for more details.


The above statement says if $foo equals 5 or 6 then execute some code.
