Can someone explain why the below switch matches the 0 element?

$ cat switch.php
<?php

$tmp = array(
        'foo' => 1,
        'bar' => 2,
        'zap',
);

foreach ($tmp as $key => $val) {
        switch ($key) {
                case 'foo':
                case 'bar':
                        echo "[$key][$val]\n";
                        break;
        }
}

$ php -f switch.php
[foo][1]
[bar][2]
[0][zap]
