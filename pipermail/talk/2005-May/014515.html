Tom,

I run PHP (CGI version) and MySql in the PASE environment on the
AS/400. PASE does not require an additional Linux partition. PASE
supports AIX binaries. I run Mambo and PHPAdsNew there.

The performance is mediocre, so I wouldn't try it with a high-volume
site. IBM states that they will improve support for PHP by the end of
2005.

This web site is helpful: i5php.net
