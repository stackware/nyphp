Hey dude:

The following code looks for stuff in the format of:

www.----

and replaces it with 

<a href = 'http://www.----'>www.-----</a>

You can tweak it for your purposes.

Good luck,

Hise

---------------------------------
<?php
$test_text = ereg_replace("www.[^<>[:space:]]+[[:alnum:]/]", "<a href =
http://\\\\0>\\\\0</a>", $test_text);

?>
---------------------------------
