If you go with Mandrake 8.2 on an Inspiron (mine is inspiron 8100 with the nvidia chip) everything will be probed and configured properly except for the modem (for which I am told there's a lucent driver) and XFree. You need nvidia's own kernel module and driver from their site. It is pretty easy to install and use a readily available XFree configuration file to get the display going at full resolution. linux-laptops.net has around 10 pages that relate to installing on the inspiron 8100 (other models too but they are all similar).

All in all inspiron is a perfect linux laptop and if you're a newbie, you can't go wrong with Mandrake.
