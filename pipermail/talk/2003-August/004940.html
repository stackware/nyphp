I have been bitten so many times by this that I 
prefer this in my code. I find that its more 
explicit.

$bar = 0;
if ( strlen ($bar) == 0 ) {
   echo '<br />bar equals empty or not set';
} else {
	echo '<br />bar not empty';
}

Output: '<br />bar not empty'

#$bar = 0;

   echo '<br />bar equals empty or not set';
} else {
	echo '<br />bar not empty';
}
