What kind of camera do you plan to use?  If it is an IP camera (as in 
network addressable, embedded OS, web server onboard), an http 
request can be sent to the camera that returns a still image.  I 
contract for a company on Long Island that does this type of work.

If it is a USB camera, search for webcamXP -- it is a pretty decent ap 
and I think the registered version does what you need.  It's Windows 
only and is cheap (~$35 last time I checked).
