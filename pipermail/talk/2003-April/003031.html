One word of warning for the table renaming, it relies on specifics of the
SQL Server in use, ie. mySQL.  This would mean that If you were to change
out your DB Server in the future, you are not assured that your app will
still work without tweaks.  If no plans are to ever change the SQL Server
then the renameing is probably the simplies solution.
