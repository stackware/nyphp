Jon, here are some clues:

Fichier = File
Feuille = Sheet
Cellule = Cell
Connexion = Connection
Classeur = Folder
Valeur = Value
Enregistrer = Save
Fermer = Close
OuvreLeFichier=OpenFile
EcritDansCellule=WriteInCell
NouvelleValeur = NewValue
RecupereValeurCellule=ReadCellValue
EcritDansCellule=WriteInCell
