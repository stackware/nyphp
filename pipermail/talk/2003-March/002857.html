SEATTLE, Wash. – (March 25, 2003) – MySQL AB, developer of the world's most popular Open Source database, today
announced that the MySQL database version 4.0 has been labeled "production," indicating that it is ready for any MySQL
deployment. To reach production status, MySQL software must pass a series of rigorous tests with no fatal bugs and then
undergo “battle-testing” in live environments for a minimum of three months. Starting with the newest release, 4.0.12,
MySQL version 4.0 is now the standard code base for all MySQL database downloads.
