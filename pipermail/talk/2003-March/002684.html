Hey, are you Eddy Nunez from NJIT?

Solving the guys' problem:

$Child is type Array.
To see the data, here is an example to use (as dynamic2.php):

<?
foreach ($_GET['Child'] as $name) {
        print "$name<br>";
}
?>

You should really use a POST instead of a GET for dynamic.php.
With lots of children, the url could get really messy.
