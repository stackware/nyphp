>>user table in MySQL to authenticate parents.

My answer to this is, well, yes and no ;^)

I've done it with Zope/Plone, but never from PHP.  The one thing that I 
remember is that there are two ports that an AD server listen to for LDAP, 
the 'official' one and the other for something like 'data services' or 
somesuch.  It is this second port that your PHP apps will have to bind to, 
as the 'official' port mangles all transmissions to an almost unusable 
state.
