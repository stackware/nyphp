Andrew:

You can keep it static, but the child class must manually modify the 
values as desired rather than setting the property in the child.

class a {
    public static $aa = 'original';
    public static function e() {
        echo self::$aa . "<br/>\n";
    }
}

class b extends a {
    public function m() {
        self::$aa = 'modified';
    }
}

b::m();
b::e();
