There is also a great free tool - tidy -  that coverts HTML to XHTML.
So you don't *actually* have to concern yourself with writing XHTML.

http://www.w3.org/People/Raggett/tidy/ 

Tidy can even chew through MSHTML, the junk you get when you Save As
Webpage in Word. So you can write using whatver tool you want. You can
also compile PHP with support for libtidy if you want to script it.

The only drawback with markup for a document with screenshots is that
you have to put the screenshots on a public server somewhere.
