Macromedia designed Flash MX Remoting for use as 
a direct transfer link between Flash MX and 
Cold Fusion MX. 

The protocol used in Flash MX Remoting is called 
ActionScript Message Format (AMF).

The AMF format used in remoting is binary, and so 
is more compact than XML -- hence (it can be) more 
efficient. 

Some clever guys at: 

http://www.amfphp.org/ 

(the flash remoting for php site)

figured out the format. 

There are some tutorials on flash remoting for php 
along with some sample programs there.
