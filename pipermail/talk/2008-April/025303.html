A number that cannot by divided by any other number except 1 and itself.

For your manager... at the top of my head, a small piece of code
(written in normal algorithm form):

declare match

for i := 1 to N do
    match := 0

    for j := 1 to i do
        if i mod j = 0 then
            match := match + 1
        end if
    end for

    if match <= 2 then
        print N
    end if
end for


So, given 10 will return you:

1, 2, 3, 5, 7
