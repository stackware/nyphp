> André Pitanga wrote:
>> 4) Which is the correct CSS syntax?
>>      a) body {color: black}

>>      c) {body:color=black}
>>      d) {body color:black}

> Doesn't correct CSS syntax require a semicolon?
> body {color: black;}

CSS only requires the semi-colon when there is more than one statement  
in the block.

This is ok:
body {
    color: black
}

This is not ok:
body {
    color: black
    background-color: white;
}
