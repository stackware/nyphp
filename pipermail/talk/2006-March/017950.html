I would check that the mail server and dns on server 2 is actually 
correct and that the mail isn't being eaten by spam filter somewhere.  
If you have a bad return-path address I have run into this problem.  Or 
if you are sending from an address that doesn't exist or if the server 
doesn't have a reverse lookup address.
