The 7th parameter (yes 7 of 8) to imagettftext() is 'The path to the
TrueType font you wish to use', look for any files on your machine
ending with the .ttf extension, those are the font files. Either
supply the full path to the file you want to use, or make your life
easier by copying that file into the same directory as the code.
