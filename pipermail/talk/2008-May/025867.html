For #1, if you had parentheses around what you want together, MySQL will understand what to return.  For example (no logic here, just example):
SELECT * FROM table WHERE ( this=this ) AND ( ( that=that OR 
thing=thing ) OR ( ( blah=blah OR foo=foo ) AND ( bar=bar OR etc=etc ) ) )
