> As you can see, its loading the file (index.php) 5 times for a  
> single request. (the hash, style, favion, png and favicon)..

Seems like you are probably using relative paths to link to those  
resources...

like url(pc_app_logo.png) vs an absolute path (/pc_release/ 
pc_app_logo.png)

So those requests are going to http://192.168.1.102/pc_release/ 
index.php/style.css, http://192.168.1.102/pc_release/index.php/ 
favicon.ico, etc...

That's the one minor inconvenience with using these types of  
'friendly' urls.
