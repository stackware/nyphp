I'd used RadLinks (http://www.radinks.com/downloads/upload.php) in the past
for a couple projects with reasonable success, though the process of loading
(and confirming) a java application can really send inexperienced users for
a loop.  The worst end being that it wouldn't load in the browser Every time
(loaded properly about 80% of the time by my completely estimated
calculation).

On an up-and-coming project, I've been considering flash's newly gained file
upload capabilities.  Here's an interesting example:

http://www.tink.ws/blog/files/flash8/FileReferenceExample.html

Adobe's tutorial (using php):

http://www.adobepress.com/articles/article.asp?p=464427&rl=1

I'm guessing it might be easier to use and especially customize, but then
again I have a Lot more flash experience than Java.

Good Luck!
