Did not look at your actual script, but... 

Certain scripts actually need to be loaded just before the closing body tag
like so, 
so that for sure all the elements needed are already available on the page.

<script></script>
</body>

Alternatively, sometimes you may also need to put your first default call to
the script functions in the same 
Place, like so:


<script>
funtion_call();
</script>
</body>
