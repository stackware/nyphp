Use Vim :-)

Fire up X11 and use an xterm.  It is far better than the Terminal app
IME.  And you could also get into any of the other X-based terminal apps
if you want.

Vim 7.0 has *awesome* remote if you choose to run it locally.  Though
you may need to build it from source... dunno.  Haven't fired up my mac
in a year.  Otherwise, you'll likely be using Vim 6.X on the remote
server, which is, of course, very good, too.  Either way, try X11 and
xterm.
