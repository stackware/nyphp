Recently did a pretty exhaustive survey of JavaScript HTML editors, I
found that YUI-editor to be the best and most actively developed, I'm
not sure if it natively supports Word Cut + Paste though:

http://developer.yahoo.com/yui/editor/

A note: All JS HTML editors have different quirks for each browser.
Before moving anything to production, I highly recommend doing an
exhaustive round of testing and allowing your users to try the editor
out as an "In Testing" feature.
