I believe you would need to put the HTML tabs inside the heredoc to see the
linebreaks.  The heredoc puts it straight into the HTML.

I think you would need:

<?php

echo <<<EOD
<br />
This is a heredoc test.<br />
<br />
Test line 2.<br />
    Test line 3 with tab.<br />
<br />
EOD;
?>
