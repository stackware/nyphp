On Fri, 5 Sep 2003, Peter Lehrer wrote:

> Does anybody know where you can get DB.php and how to install it?
> Does it just go in your include directory?

You install it using pear. See http://pear.php.net. PEAR files usually
go in /usr/local/lib/php, but the PEAR Package Manager should take
care of this for you. BTW, you may need to edit your php.ini file to
add the new location to your include_path.
