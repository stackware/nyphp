This works for me:

<?php
  $agx = array ('back_door' => '0',
                        'front_door' => '1',
                        'side_door' => '0',
                        'garage_door' => '1');
  foreach ($agx as $k => $v) {
    echo "\$agx ['{$k}'] = {$v}\n";
  }
?>

Output:
$agx ['back_door'] = 0
$agx ['front_door'] = 1
$agx ['side_door'] = 0
$agx ['garage_door'] = 1
