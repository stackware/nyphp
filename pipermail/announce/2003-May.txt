----- phpComponent Architecture Beta

New York PHP is pleased to release a beta version of the phpComponent
Architecture. A New York PHP supported Open Source project, the phpComponent
Architecture provides an efficient way to utilize remote code. Implementing a
transparent cache, pCom enforces high coding standards, versioning, packaging
and a public directory service for locating, submitting and managing
components.  See more at http://pcoms.net
